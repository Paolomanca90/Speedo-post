<!-- USAGE EXAMPLES -->
## Usage 3️⃣ <span id="usage"></span>

The site is divided into 2 sections: one dedicated to admin and one to users.
First of all you will have to create an admin in your DB. This way you will be able to log in and add delivery.
Each collaborator can insert a new shipment including the sender's personal data and the recipient's information. Each time the package is moved, its status can be changed by searching for the shipment in the general list.
A separate section instead allows you to filter shipments based on specific search criteria based on query selection.
The user, however, by entering his/her CF can view his/her shipments and the progress of the delivery.
